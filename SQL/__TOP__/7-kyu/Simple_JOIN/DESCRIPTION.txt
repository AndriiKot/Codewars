For this challenge you need to create a 
simple SELECT statement that will return 
all columns from the products table, 
and join to the companies table 
so that you can return the company name.

products table schema

id
name
isbn
company_id
price

companies table schema

id
name

You should return all product fields 
as well as the company name as "company_name".


RU:
Для этой задачи вам нужно создать простой оператор SELECT, 
который будет возвращать все столбцы из products
таблицы и присоединяться к companies
таблице, чтобы вы могли вернуть название компании.

Вы должны вернуть все поля продукта, 
а также название компании как «имя_компании».
