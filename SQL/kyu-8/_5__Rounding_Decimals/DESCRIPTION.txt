Given the following table 'decimals':

** decimals table schema **

id
number1
number2

Return a table with two columns (number1, number2), 
the value in number1 should be rounded 
down and the value in number2 should be rounded up.


RU:
Учитывая следующую таблицу «десятичные дроби»:

** decimals table schema **

id
number1
number2

Возвращает таблицу с двумя столбцами (номер1, номер2), 
значение в число1 должно быть округлено в меньшую сторону, 
а значение в число2 — в большую сторону.
