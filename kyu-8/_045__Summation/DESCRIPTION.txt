Summation
Write a program that finds the 
summation of every number from 1 to num. 
The number will always be a positive 
integer greater than 0. Your function 
only needs to return the result, what 
is shown between parentheses in the 
example below is how you reach that 
result and it's not part of it, see the sample tests.

For example (Input -> Output):

2 -> 3 (1 + 2)
8 -> 36 (1 + 2 + 3 + 4 + 5 + 6 + 7 + 8)

RU:
Суммирование
Напишите программу, которая находит 
сумму всех чисел от 1 до num. Число 
всегда будет положительным целым числом, 
большим 0. Вашей функции нужно только 
вернуть результат. В приведенном ниже примере в скобках показано, 
как вы достигаете этого результата, 
и оно не является его частью, см. примеры тестов.
