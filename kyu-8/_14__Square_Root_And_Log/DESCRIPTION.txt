** decimals table schema **

id
number1
number2

Return a table with two columns (root, log) 
where the values in root are the square root 
of those provided in number1 and the values 
in log are changed to a base 10 logarithm from those in number2.

RU:
Возвращает таблицу с двумя столбцами (корень, журнал), 
где значения в корне представляют 
собой квадратный корень из значений, указанных в номере1, 
а значения в журнале изменяются 
на логарифм по основанию 10 по сравнению со значениями в номере2.
