Task
Create a method to see whether the string is ALL CAPS.

Examples (input -> output)
"c" -> False
"C" -> True
"hello I AM DONALD" -> False
"HELLO I AM DONALD" -> True
"ACSKLDFJSgSKLDFJSKLDFJ" -> False
"ACSKLDFJSGSKLDFJSKLDFJ" -> True

In this Kata, a string is said to be 
in ALL CAPS whenever it does not 
contain any lowercase letter so any 
string containing no letters at 
all is trivially considered to be in ALL CAPS.

RU:
Строка в верхнем регистре?
Задача
Создайте метод, чтобы проверить, написана ли строка ЗАГЛАВНЫМИ БУКВАМИ.

В этом Ката говорится, что строка написана 
ЗАГЛАВНЫМИ буквами, если она не содержит 
ни одной строчной буквы, поэтому любая строка, 
вообще не содержащая букв, 
тривиально считается написанной ЗАГЛАВНЫМИ буквами.
